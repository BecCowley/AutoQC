IQuOD automatic quality control testing suite

A testing suite for automatic quality control checks of subsurface ocean temperature observations

Download this project as a .zip file Download this project as a tar.gz file

The International Quality-controlled Database (IQuOD) initiative aims to assemble the highest-quality historical subsurface ocean temperature (and salinity) public database, along with the most complete metadata and assigned uncertainties. This will be achieved by pooling resources and expertise into a single, best practice community effort.

This project aims to compare the results of automatic quality control tests on data of known quality in order to find the best set of tests to use.
